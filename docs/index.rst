Flask-Arrested
===============

Flask-Arrested is a framework for rapidly building robust REST API's on top the of the python framework Flask.
It makes no assumptions about the "right" or "wrong" way to implement REST.  Its simply provides a flexible foundation to enable you to quickly build
secure and scalable API's.
